## О проекте

Исследование поведения клиентов по кредитному продукту банка. Только 40–50% клиентов активируют карту в первые 30 дней после получения — задача найти причины и предложить решения.

**Датасет:** 2 000 клиентов, 96 000+ транзакций за 2023–2024 гг.

## Цели и задачи

- Проанализировать воронку активации кредитной карты
- Выявить сегменты с низкой активацией
- Исследовать транзакционное поведение активных клиентов
- Провести когортный анализ по месяцам выдачи
- Сформировать рекомендации по росту активации

## Методология

**1. Предобработка данных**
- Очистка и типизация данных
- Feature engineering: возрастные группы, месяц выдачи, период активации

**2. Анализ активации**
- Общий уровень активации по продукту
- Сегментный срез: Массовый / Массовый+ / Премиум / Private
- Распределение времени до первой транзакции

**3. Канальный и региональный анализ**
- Активация по каналу привлечения клиента
- Региональные различия в поведении

**4. Транзакционный анализ**
- Топ категории расходов
- Средний чек по сегментам
- LTV-потенциал по группам клиентов

**5. Когортный анализ**
- Динамика активации по месяцам выдачи карт
- Сезонные паттерны

## Ключевые выводы

- **Массовый сегмент** активирует только ~38% — основная точка роста
- **Мобильное приложение** даёт наивысшую активацию среди каналов
- **Push-коммуникации** на 1/3/7 день после выдачи могут повысить активацию на 10–15%
- **Кешбэк на топ-категории** (Продукты, Онлайн) — лучший стимул для первой транзакции
- **Летние когорты** активируют лучше зимних — нужна усиленная коммуникация в низкий сезон
